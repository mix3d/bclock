Welcome to Beautiful Clock

Beautiful clock was started like most other projects; I was looking for something and just couldn't find it. I moved locations at work, and my new desk-phone was an older model with no clock. I wanted an attractive, sleek, well designed clock to display fullscreen on my second monitor and/or my iPad, and couldnt find one, so I started this. Technically it is a clone of many other clock apps, but mine is free and open source.

Runs anywhere

Written in plain HMTL and Javascript, it is simple and lightweight, letting the design shine through. Currently it runs on most browsers and tablets, but the beauty of it lies in that it only needs a few css changes to make it fit phones. I plan on adding more features before I design for phones. It also has iphone/ipad web-app support, so it will work like a native app when you pin it to the homescreen.

Customizable

Works for both portrait and landscape themes. While there are just a few set themes now, the following features are planned and in the works:

  • User customizable style selection, including font, color, background, and maybe even clock position
  • Timed style rotation
  • Preferences stored locally
  • Weather integration
  • Feel free to message me with more ideas

Enjoy!
